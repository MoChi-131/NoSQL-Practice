# NoSQL Coursework

This folder is prepared for GitHub submission and contains the key coursework deliverables:

- `query.sql` — Cypher/Neo4j query script and data loading model queries for the coursework.
- `pipeline.txt` — MongoDB aggregation pipelines and example query logic used in the assignment.
- `C2087025-CM3104-report.pdf` — the written coursework report.
- `Dataset.zip` — zipped dataset archive for the coursework.

## Contents

### `query.sql`
- Includes graph database constraints for Neo4j.
- Contains CSV data load examples using APOC and periodic iteration.
- Defines graph model relationships and indexes.
- Includes Cypher queries for the coursework questions.

### `pipeline.txt`
- Includes MongoDB aggregation pipeline code for data transformation and output collections.
- Includes query examples for the coursework tasks and part 3 relationships.

### `C2087025-CM3104-report.pdf`
- The formal report describing the task, methodology, results, and conclusions.

## Usage

1. Extract `Dataset.zip` into your working folder or use the original `Dataset/` folder from the workspace.
2. Use the Neo4j `query.sql` script for graph loading and question queries.
3. Use the MongoDB `pipeline.txt` script for aggregation pipeline examples.

## Notes

- The `Dataset/` folder is not copied into this package to keep the submission focused on the deliverables.
- If you want the full dataset included in the repository, copy the relevant CSV/JSON files from the parent workspace.
